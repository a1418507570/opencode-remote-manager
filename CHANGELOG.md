# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and uses repository tags for version numbers.

## [0.1.0] - 2026-04-09

### Added

- Initial release of the macOS menu bar utility.
- Remote `opencode serve` bootstrap via SSH and `crontab @reboot`.
- Local tunnel supervision with SSH control sockets.
- Desired-state persistence and self-healing reconcile loop.
- Desktop-oriented localhost remote integration workflow.
